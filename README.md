# Sentinel

[![Netlify Status](https://api.netlify.com/api/v1/badges/a387ea06-18ee-4884-a973-1e36a80109a9/deploy-status)](https://app.netlify.com/projects/sent1nel/deploys)

Sentinel is an **intelligent danger detection and reporting platform** designed to enhance community safety through real-time information sharing. Users can report incidents, view reports on a map, and access safety alerts to stay informed about their surroundings.

## **Features**

- **Real-time incident reporting** with forms and geolocation.  
- **Interactive map** showing reported incidents using Leaflet.  
- **Firebase integration** for secure authentication and database storage.  
- **Responsive UI** for desktop and mobile users.  
- **Deployment on Netlify** for easy access.  

---

## **Tech Stack**

- **Frontend:** React, React-Leaflet, CSS  
- **Backend / Database:** Firebase Firestore  
- **Authentication:** Firebase Auth (Google Sign-In)  
- **Hosting:** Netlify
